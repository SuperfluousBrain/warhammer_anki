40k List to Anki Converter

What is it

This page converts a list that is in New Recruit's json format to flash cards that are usable by the Anki flashcard program. Memorizing your list's stats will save you clock time. Anki is the best flashcard program for the job.

How to use it.

  1. Go to New Recruit.
  2. Make a 40k list containing the models with the weapon profiles you want to study.
  3. Export the list as .json.
  4. On this page, choose the .json file to convert.
  5. Click dowload. This will create a .txt file.
  6. Import that .txt file into anki.
